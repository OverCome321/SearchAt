# SearchAt
### SearchAt Flutter/Dart (Огромный проект разрабатывался в команде с одногруппником на протяжении месяца). Приложение разрабатывалась под Android (под IOS не оптимизировано). Здесь есть возможность создавать и делится своими идеями и если кому-то понравится то что вы придумали у вас есть возможность реализовать это вместе (например вы не умеете создавать сайты, но у вас есть крутая идея для этого, вы можете разместить идею в SearchAt и найти с кем ее реализовать). Приложение поддерживает локализацию на 2 языках Eng Rus, в зависимости от локализации вашего смартфона будет выбран язык приложения.
